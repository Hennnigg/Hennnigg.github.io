### **반복문**
 자동화는 반복과 분기로 끝난다.



### **함수 def**
#### mydef01 : 사칙연산 메서드
```Python
def add(a,b):   
    return a + b

def minus(a,b) :
    return a - b

def multiply(a,b) :
    return a * b
 
def divide(a,b) :
    return a / b


sum = add(4,2)
min = minus(4, 2)
mul = multiply(4 ,2)
div = divide(4 ,2)

print("sum = ", sum) #6
print("min = ", min) #2
print("mul = ", mul) #8
print("div = ", div) #2.0
```
